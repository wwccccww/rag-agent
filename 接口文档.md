请求方式：`POST`

参数说明：

#### 3.6 查看步数排行榜

请求路径：`/user/rankings`

请求方式：`GET`

接口描述：查看日/周/月步数排行榜

参数格式：`query string`

参数说明：

| 名称         | 类型     | 是否必须 | 备注                                 |
|:---------- |:------ |:---- |:---------------------------------- |
| periodType | string | 是    | 周期类型：day（日排行）、week（周排行）、month（月排行） |
| pageNum    | number | 非必须  | 页码，默认1                             |
| pageSize   | number | 非必须  | 每页条数，默认10                          |

##### 3.6.3 响应数据

参数格式：`application/json`

参数说明：

| 参数名                | 类型       | 是否必须 | 备注              |
| ------------------ | -------- | ---- | --------------- |
| code               | number   | 必须   | 响应码，1代表成功，0代表失败 |
| msg                | string   | 非必须  | 提示信息            |
| data               | object   | 非必须  | 返回数据            |
| \|- total          | number   | 非必须  | 总记录数            |
| \|- list           | object[] | 非必须  | 排行榜列表           |
| \|- \|- rank       | number   | 非必须  | 排名              |
| \|- \|- nickname   | string   | 非必须  | 用户昵称            |
| \|- \|- avatar     | string   | 非必须  | 头像URL           |
| \|- \|- totalSteps | number   | 非必须  | 周期内总步数          |

| \|- totalSteps     | number   | 非必须  | 统计周期内总步数        |
| \|- averageSteps   | number   | 非必须  | 统计周期内日均步数       |
